# 🧠 Review Sentiment Analyzer

A Machine Learning-based desktop application developed using **Python**, **CustomTkinter**, and **Scikit-learn**. The application analyzes customer reviews and predicts whether the sentiment is **Positive** or **Negative**.

---

## ✨ Features

- 🔍 Analyze customer reviews
- 😊 Predict Positive or Negative sentiment
- 📊 Confidence score with progress bar
- 📄 Detailed Analysis Report
- 🌙 Light/Dark Theme Switch
- 📈 Analysis Statistics
- 📝 Recent History
- 💾 Save Results
- 🔄 Reset Statistics
- 🎨 Modern CustomTkinter GUI

---

## 🛠 Technologies Used

- Python
- CustomTkinter
- Scikit-learn
- Joblib
- Pandas
- TF-IDF Vectorizer
- Multinomial Naive Bayes

## 🚀 Installation

1. Clone the repository:

```bash
git clone https://github.com/sidrasamad330-del/Review-Sentiment-Analyzer.git
```

2. Open the project folder:

```bash
cd Review-Sentiment-Analyzer
```

3. Install the required libraries:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python gui.py
```

## 📚 Machine Learning Model

- Algorithm: Multinomial Naive Bayes
- Feature Extraction: TF-IDF Vectorizer
- Task: Binary Sentiment Classification
